Betrayal is a map redrawn
while you were still living in the old country.

Sleep becomes evidence collection.
Dreams subpoena the loss.

Strength, if it appears,
is not triumph —
it is getting vertical
without calling vertical healing.

Fresh start is optional.
So is never trusting the same door twice.

Love may lead astray.
That is not the same as saying
you deserved the detour.

The map still shows the old border.
